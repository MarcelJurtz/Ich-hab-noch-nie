# Ich hab noch nie...

App zum bekannten Trinkspiel.

Aktuell sind ca. 350 Texte vorhanden, es besteht die Möglichkeit, eigene hinzuzufügen, sowie zwischen Eigenen Texten / Vorgebenenen Texten / Beiden Kategorien zu filtern.
